# Merigate Landing Page Project

## Goal
Build a polished, modern, responsive landing page for www.merigate.com.

## Deployment Target
Cloudflare Pages.

## Website Type
Static-first marketing website. No backend, no database, no authentication.

## Allowed Tech
Prefer one of:
- Astro
- Vite + React
- Plain HTML/CSS/JS

Choose the simplest option that produces a clean, professional, maintainable result.

## Design Direction
Professional, premium, calm, trustworthy.
Merigate should feel like an AI-era education/advisory platform.
Use the provided logo, brand images, and content.
Preserve the Merigate flavor: global, intelligent, structured, future-facing, education-oriented.

## Required Sections
- Hero section
- Trust / positioning statement
- Services or offerings
- Why Merigate
- Lead capture / inquiry form
- Footer

## Form
For now, create a frontend-only form UI.
Do not connect to a backend yet.
Prepare the structure so it can later connect to Formspree, Google Forms, Cloudflare Workers, or n8n.

## Constraints
- Must be mobile responsive.
- Must deploy cleanly on Cloudflare Pages.
- Avoid unnecessary dependencies.
- Do not add backend services.
- Do not require paid services.
- Do not expose secrets or API keys.
- Optimize images for web performance.
